# Rock Paper Scissors Game

This is a classic Rock, Paper, Scissors game implemented with HTML, CSS, and JavaScript, created as a project for The Odin Project curriculum.

## Features

- **Interactive UI**: Play against the computer by clicking on icon-based buttons for Rock, Paper, or Scissors.
- **Round-based Gameplay**: The game is played over 5 rounds.
- **Countdown Timer**: A 3-2-1 countdown adds suspense before each round's result is revealed.
- **Score Tracking**: The scores for both the player and the computer are displayed and updated in real-time.
- **Clear Winner Declaration**: At the end of 5 rounds, a final winner is declared.

## How to Play

1. Clone or download this repository.
2. Open the `index.html` file in your web browser.
3. Click on the Rock, Paper, or Scissors icon to make your choice.
4. The game will play out for 5 rounds and declare a winner.
5. To play again, simply refresh the page.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
